## React Table App
This app focus on search filtering, button filtering and pagination.

## Purpose
I built this mini project to practice implementing filter and pagination on large database

### Core Feature
- Realtime name search
    - Search name based on user input including lowercase or uppercase 
- Dynamic button filtering 
    - Quickly categorize data by its tag eg.role manager, role admin etc
- Pagination  
    - Separate a large data to smaller group per page

### Tech Used
- **Framework**: React 19 with compiler
- **Styling**: Tailwind CSS
- **Language**: Typescript

### Library Used
- [Floating UI](floating-ui.com)
- [React Icon](react-icons.github.io)
