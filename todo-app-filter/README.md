# TODO APP

A simple task management application focused on efficient state management, array mutations, and dynamic sorting.

### Purpose

I built this app to and implementing sorting, add task, and delete task. Using filtration array knowledge on my previous project. This project aim to implementing robust logic on adding, deleting, and sorting task.  

### Core Feature
- Search by task: Instantly find task by title
- Task Management : Seamless add and delete task function
- Filter by Tag : Organize task based on specific categorize
- Dynamic sorting: Reorder task based on the completion status 

### Tech Used
- **Framework**: React 19 with compiler
- **Styling**: Tailwind CSS
- **Language**: Typescript
